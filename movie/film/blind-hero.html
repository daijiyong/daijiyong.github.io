          目中无人：一把听风刀，成瞎子睁眼就要杀人

          Mr.Dai2022年7月5日
          • 电影
          大约 3 分钟

          导演: 杨秉佳

          编剧: 杨秉佳

          主演: 谢苗 / 高维蔓 / 向皓 / 刘奔 / 张荻 / 张皓森 / 高毅 / 刘军 / 康轩 / 侯骏桀 / 尹箭 / 甘书杰 / 张乐 / 张韦 / 王军 / 王奕彤 / 闫宁

          类型: 动作 / 武侠

          制片国家/地区: 中国大陆

          语言: 汉语普通话

          上映日期: 2022-06-03(中国大陆网络)

          片长: 74分钟

          又名: 盲侠

          欢迎公众号内回复“目中无人”交流讨论

          落寞将军、身怀绝技的盲眼“捉刀人”成瞎子，遇上惨遭灭门和凌辱的酒家女倪燕，原本不想插手的他逐渐被卷入一场势力庞大的暗黑纷争中，成瞎子开始踏上了寻求公道的复仇之路

          ##1

          没错，片中主演就是他：谢苗

          被誉为李连杰接班人的武打童星

          只不过长大之后就再没有什么热度了

          最近几年开始陆续拍了很多网络电影，渐渐地回到了大众的视野中

          ##2

          开篇就满满的武侠江湖气

          近些年武侠电影一直在没落

          一方面因为武打明星青黄不接，后继无力

          另一方面就是流量当道，全是小鲜肉，看到那张脸就实在没办法跟武侠阳刚之气联系上

          ##3

          作为一部小成本的网大

          时长仅仅75分钟

          但是也庆幸没有为了凑90分钟而强行加戏

          小成本精品武侠，动作和台词都挺干脆

          虽然剧情略显老套，但是动作设计可称上乘

          ##4

          一双错骨手

          一把听风刀

          一招楼兰斩

          关二爷一睁眼要杀人成瞎子也是

          ##5

          英雄向来难过美人关

          不让大家失望所以男主必须要有爱情故事

          但是剧中没有往日让人反感的拖拉和腻歪

          女二的牺牲没有太多波折

          英雄救美也没有上演反而更让故事情节合理且完整

          ##6

          武打戏是这部电影好评最重要的因素

          男主没有任何谋略智慧

          只凭一身武艺以一当十

          团灭了整个黑恶势力

          非常过瘾

          有些网友也尽量不要抬杠了

          正如吴京说的：美国人一人干一个师就科学，中国人一人打十几个人咋就不行？

          ##7

          事了拂衣去，深藏功与名

          电影节奏也是如此

          最终男一和女一也并没有俗套的在一起

          很多人说这是在翻拍《座头士》，就开喷

          我是没看过《座头士》，只看过《目中无人》

          如果每个国产翻拍剧都能翻拍成这个程度，那估计也没多人在意原片
