                        青瓷：伪君子的商道

                        大约 4 分钟

                        导演: 李骏

                        编剧: 浮石

                        主演: 王志文 / 张国立 / 韩雨芹 / 王海燕 / 杜江 / 李菲儿 / 李洪涛 / 章申 / 郑晓宁 / 刘敏 / 叶一云

                        类型: 剧情 / 爱情

                        制片国家/地区: 中国大陆

                        语言: 汉语普通话

                        集数: 40

                        单集片长: 45分钟

                        又名: Green Porcelain

                        欢迎公众号内回复“青瓷”交流讨论

                        一部老电视剧了，但是话题比较冷门，所以看过的人估计不多，讲述的是关于雅贿的话题。王志文和张国立的演绎更让这部剧上了一个档次，非常值得一看。

                        #

                        ##1

                        本剧演绎是一个敏感的话题，核心是“雅贿”

                        讲述的是资产拍卖行业中，一个拍卖公司与法院、与资产管理公司周旋博弈的故事

                        虽然部分剧本和部分情节着实拉跨、制作稍显廉价，但好在王志文和张国立等老戏骨的演技在线，所以绝不失为一部上乘的国产电视剧

                        #

                        ##2

                        张国立饰演反派的时候不多，印象比较深的是挑起六大门派纷争的圆真大师，但是看这张脸怎么都不像反派

                        但是在《青瓷》中却把颜若水的狠辣、狡猾演绎的入木三分。

                        #

                        ##3

                        剧中有两场非常精彩的男一和反一对手戏，分别在第5集和第47集

                        前者表现的是颜若水作为权力拥有者向下讲述“爷孙规矩”

                        后者表现的是张仲平撕破脸皮反攻强权

                        两位老戏骨对台词、情绪和表情的控制都十分精妙，代入感极强

                        #

                        ##4

                        徐艺作为一名重要的反派，太逊了！！！

                        用颜若水一句话就是“大事上犯糊涂、小事上耍聪明”

                        不仅贪得无厌，而且卑鄙下流，想出来的所有办法都是断子绝孙的一锤子买卖，不给别人留后路的时候也把自己的后路给堵死了

                        如果不是张仲平念在跟徐艺是姨甥关系而处处袒护，真的是过不过三集那种，中间差点因为这家伙太过气人看不下去了

                        #

                        ##5

                        伪君子的商道

                        这可能是对张仲平最精准的概括了

                        虽然张仲平左右逢源，事事追求双赢，但是其实也只是对自己违法乱纪的虚伪包装

                        虽然自认为一直恪守原则，但其实也是看人下菜碟，不停的突破底线

                        他是从不害人，却在破坏规则

                        他是忠于家庭，却也会心猿意马

                        他是重情重义，却也在利用朋友的性情，不够坦率

                        #

                        ##6

                        丛林是剧中唯一一个始终坚守规则、讲道理、恪守底线的人

                        有点太正，但不让人反感，没有做作的感觉，演绎的恰到好处

                        相比《人民的名义》中的侯亮平，更像一个活生生的人

                        家庭也许一地鸡毛，事业也许跌跌撞撞，履历背景也是平庸至极，无论跟谁聊天都是话题终结者

                        我们每个人都希望所有的法官都像丛林和侯昌平，但是我们每个人又绝对不愿成为他们那样的人

                        #

                        ##7

                        曾真，作为剧中的女一，其实最大的问题其实就是选角不够漂亮

                        #

                        ##8

                        鲁冰在整个剧情其实是一个非常关键的人物，他是能跟所有人都扯上关系的角色，但同时也是活得最累的一个人：

                        恪守一定的原则但不及丛林

                        能够左右逢源但不及张仲平

                        对权力和金钱有欲望且有一定的手段但不及颜若水

                        有极强的关系和背景但又不敢利用
